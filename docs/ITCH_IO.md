# CuteSpace - itch.io Page Draft

## Short Description

A floating Windows bubble for modes, shortcuts, routines, clipboard history, and focus time.

## Description

CuteSpace is a small local Windows app that keeps a soft floating bubble on your desktop. Open it to organize modes, launch apps or websites, keep quick shortcuts nearby, review recent clipboard items, and use a simple focus timer or counter.

This is the first public release, built for Windows x64.

## Install

1. Download `CuteSpace-v0.1.0-windows-x64.zip`.
2. Extract the zip.
3. Run `CuteSpace.exe`.

Windows may show a warning because the app is not code-signed yet.

## Privacy

CuteSpace works locally and does not upload your data. Modes, shortcuts, clipboard history, preferences, and logs stay on your PC.
